## Pasos para configurar Webpack

### Instalaciones.

1. Iniciar un proyecto de Node.js (Se genera el package.json)
   
   ```sh
    npm init -y 
   ```
2. Ahora instalamos el webpack y webpack-cli
   
   ```sh
   npm install webpack webpack-cli --save-dev
   ```
